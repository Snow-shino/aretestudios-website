Projection Modes

TileForge2D keeps gameplay grid-based and projection-aware. Projection controls world placement, editor picking, rendering layout, and sorting.

  • TopDownOrthographic
  • IsometricDiamond
  • IsometricStaggered later

Gameplay systems should stay grid-based. Only world positioning, mouse picking, rendering, and sorting should depend on projection mode.
