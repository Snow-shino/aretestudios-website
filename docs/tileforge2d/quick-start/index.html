TileForge2D Docs

Quick Start

  1. Create a TileForge Tileset.
  2. Assign a source texture, set tile size, and sync tile metadata.
  3. Create a TileForge Map.
  4. Add the tileset to the map asset.
  5. Open the map, select a tile, and paint on the active layer.
  6. Place a TileForgeMapActor in a level and assign the map asset.
