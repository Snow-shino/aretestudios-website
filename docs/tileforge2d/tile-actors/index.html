Tile Actors and Smart Tiles

The map owns grid placement. The actor owns gameplay logic.

Use Tile Actors for doors, chests, pots, signs, switches, breakable rocks, cuttable grass, pickups, and warp points.

  1. The player asks the map for the current or targeted grid cell.
  2. The player sends an interaction tag.
  3. The tile actor validates the interaction.
  4. The actor plays effects, changes state, destroys itself, or asks the map to replace a tile.
