TileForge2D

TileForge2D is a professional Unreal Engine editor plugin for game-ready 2D tile maps, layers, collision, Tile Actors, destructibles, projection modes, and Blueprint hooks.

Core workflow

Tileset asset -> Map asset -> Paint layers -> Generate map actor -> Layer collision -> Destroy tile.
