## 1.0.0

Initial release.

### Added

- Ase2 Source asset workflow.
- Aseprite command-line export integration.
- Paper2D spritesheet texture import.
- PaperSprite generation.
- PaperFlipbook generation from Aseprite frame tags.
- Grid extraction mode.
- Auto from Aseprite JSON extraction mode.
- Marker layer import for `SOCKET`, `HIT`, and `HURT` layers.
- PaperSprite socket writing from 1x1 marker pixels.
- Optional advanced hitbox data asset.
- Stable reimport behavior for generated assets.
- Import report sections for **What was done**, **Notes**, and **Warnings**.

### Requirements

- Unreal Engine 5.
- Paper2D enabled.
- Aseprite installed locally.
