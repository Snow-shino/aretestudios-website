Ase2UE is an Unreal Engine editor plugin for bringing Aseprite art into Paper2D projects.

Use this documentation to install the plugin, import your first file, understand Grid vs Auto import modes, create flipbooks, and add sprite sockets from marker layers.

## Chapters

### 1. Getting Started

Install the plugin, enable Paper2D, set the Aseprite executable path, and create your first `Ase2 Source` asset.

[Open Getting Started](./getting-started/)

### 2. Importing

Understand the `Ase2 Source` settings, import destination, Grid mode, Auto from Aseprite JSON, flipbooks, frame durations, and reimport behavior.

[Open Importing](./importing/)

### 3. Hitboxes, Hurtboxes & Sockets

Use Aseprite marker layers such as `SOCKET_Hand`, `HIT_WeaponStart`, and `HIT_WeaponTip` to create PaperSprite sockets and optional Blueprint-readable marker data.

[Open Hitboxes, Hurtboxes & Sockets](./hitboxes-hurtboxes-sockets/)
