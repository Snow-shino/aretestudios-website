TileForge2D is a professional 2D world-building plugin for Unreal Engine.

The guiding principle is:

```text
Tiled-style editing, Unreal-native runtime.
```

TileForge2D is not meant to be only a visual tile painter. The goal is a game-ready map workflow with tileset editing, map assets, paint layers, collision generation, sprite sorting support, Tile Actors, destructible runtime tiles, and Blueprint gameplay hooks.

## Current Foundation

- Custom `TileForge Tileset` and `TileForge Map` assets.
- Custom tileset and map editors.
- Layer-based map painting.
- Tile metadata, collision presets, destructible settings, tags, and custom properties.
- Runtime `ATileForgeMapActor`.
- Blueprint helpers for grid, world, tile, destruction, interaction, and validation workflows.
- Projection-mode architecture for top-down and isometric maps.
- Tile Actor placement foundation for interactive doors, chests, pots, signs, pickups, and other grid-bound gameplay.
