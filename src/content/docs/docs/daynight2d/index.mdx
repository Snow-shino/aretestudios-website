DayNight2D is a foundational Unreal Engine plugin for time-driven 2D games.

It is designed for platformers, top-down games, isometric games, pixel art RPGs, metroidvanias, adventure games, survival games, simulation games, cozy-style projects, and any project that needs reusable time-based behavior.

## Current Foundation

- Timer-driven world time subsystem.
- Time settings data asset.
- Blueprint API for time, date, formatting, queries, pause/resume, speed, advancing, skipping, save/load, and replication state.
- Schedule assets.
- Routine assets and routine component.
- Timed behavior, timed progress, respawn, and availability components.
- Optional visual profiles and tint/overlay/interior components.
- Replicated time actor with server RPCs and multicast day/season/schedule events.
- Sleep vote component.
- Save/listener/schedule/routine/visual/progress interfaces.
- Gameplay tags and console commands.
